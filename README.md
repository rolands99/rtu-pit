# Programmatūras apraksts
Grafisks lietotāja interfeiss, kurā ir iespējams: <br>
* Pievienot oriģināltekstus datubāzei;
* Pievienot tulkojumus izvēlētam tekstam un valodai;
* Ir iespējams arī rediģēt (dzēst, atjaunināt) tekstus un tulkojumus.
